Particle System in a Vector Field

  • C - Clear particles
  • R - Reset particles' initial positions
  • V - Toggle vectors
  • G - Toggle grid
  • A - Toggle coordinate axes
  • S - Toggle particle visibility
  • Spacebar - Start/stop animation
  • Mouse click - add single particle to the system
  • Mouse drag - add multiple particles to the system
  • Mouse scroll - increase/decrease velocity
